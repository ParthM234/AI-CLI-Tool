# AI CLI Tool

## What this project does
A command line AI tool powered by Google Gemini that can:
- Summarize any text or txt file
- Analyze sentiment of text
- Chat with memory
- Answer any question

## How to run
1. Clone the repo
2. Install library: `pip install google-genai`
3. Get a free Gemini API key from aistudio.google.com
4. Add your API key in the code
5. Run `python aitool.py`

## Libraries used
- google-genai
- os
